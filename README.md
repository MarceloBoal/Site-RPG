🎲 RPG Web Project
Um site de RPG criado para apoiar a campanha que estou mestrando, servindo como plataforma digital de organização, narrativa e mecânicas de jogo. Além do uso prático no RPG, este projeto também funciona como portfólio, demonstrando habilidades de desenvolvimento web, organização de código e boas práticas.

📌 Visão Geral
Esse site é uma aplicação web voltada para mestres e jogadores de RPG de mesa. O objetivo é centralizar informações importantes da campanha, como personagens, mundo, regras e sistemas auxiliares, tornando a experiência mais imersiva e organizada.

O projeto foi pensado para ser modular, escalável e de fácil manutenção, permitindo a adição de novas funcionalidades ao longo do tempo.

🚀 Funcionalidades
📜 Página inicial com introdução ao mundo e à campanha
🧙‍♂️ Gerenciamento de personagens (atributos, status, história)
🌍 Informações sobre o universo do RPG (lore, mapas, facções, NPCs)
(Novas funcionalidades serão adicionadas conforme a evolução da campanha e do projeto)

🛠️ Tecnologias Utilizadas:

HTML5 – Estrutura semântica das páginas
CSS3 – Estilização, layout responsivo e animações
JavaScript – Lógica, interatividade e manipulação de dados
Banco de dados
Frameworks front-end

(Opcional / Futuro):

LocalStorage / IndexedDB
API REST
